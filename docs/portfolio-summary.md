# 포트폴리오 설명 초안

## 한 줄 소개

HTML/CSS 학습 초기에 제작한 안산 문화관광 정적 웹사이트로, 공공기관형 관광 안내 사이트의 메인/서브 페이지 구조와 반응형 레이아웃을 구현했습니다.

## 프로젝트 설명

안산의 관광명소, BIG5 가치, 시티투어 정보를 소개하는 정적 웹사이트입니다. 메인 페이지에서는 대표 관광지와 여행 정보를 카드형 섹션으로 배치했고, 서브 페이지에서는 좌측 메뉴와 본문 콘텐츠 구조를 사용해 공공기관 사이트에서 자주 보이는 정보 탐색 흐름을 연습했습니다.

첫 개발 프로젝트였기 때문에 백엔드나 빌드 시스템 없이 HTML/CSS 중심으로 제작했습니다. 현재는 포트폴리오 공개를 위해 실행 방법, 구현 범위, 한계, 개선 계획을 문서화하고 깨진 CSS 참조와 명백한 마크업 오류를 정리했습니다.

## 담당 작업

- 메인 페이지 비주얼 영역 및 관광 콘텐츠 섹션 마크업
- 관광명소/테마여행 서브 페이지 레이아웃 구성
- GNB, LNB, 푸터, 관련 배너 영역 구현
- 로컬 폰트와 이미지 자산 연결
- CSS hover/radio 기반 인터랙션 및 반응형 스타일 작성
- 포트폴리오 공개를 위한 README와 프로젝트 범위 정리

## 사용 기술

- HTML5
- CSS3
- Responsive CSS
- Local font assets

## 배운 점

- 공통 레이아웃을 여러 페이지에 반복 적용하면서 헤더, 내비게이션, 푸터 구조를 이해했습니다.
- 이미지 경로, CSS 파일 참조, 폰트 연결처럼 정적 웹사이트에서 기본이 되는 자산 관리의 중요성을 배웠습니다.
- 페이지를 완성하는 것뿐 아니라 실행 방법과 미완성 범위를 문서화해야 프로젝트가 전달 가능해진다는 점을 확인했습니다.

## 개선 계획

- 공통 헤더/푸터 중복 제거
